# Microsoft-Azure---Localizando-Servicos-por-Categoria-DIO

A Microsoft Azure oferece centenas de serviços na nuvem (como máquinas virtuais, bancos de dados, redes, IA, etc.), Usando o Portal do Azure , Acesse: https://portal.azure.com
Na barra de pesquisa superior, digite o nome do serviço (ex: Virtual Machines, Storage Accounts, App Service).
Também é possivel clicar em “Todos os serviços” no menu lateral esquerdo para ver uma lista completa, organizada por categoria.
